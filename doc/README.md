## 快速开始

1. **阅读 [通用结构与系统概览](./general.md)**，了解系统基本概念
2. **选择组件类型**：QML（高性能）或 HTML（Web 技术栈）
3. **阅读对应的开发文档**：QML 组件参考 [QML组件开发](./QML.md)
4. **使用小组件编辑器**创建新项目
5. **开始开发**你的第一个小组件！

## 开发工具

- **小组件编辑器**：创建、编辑和管理小组件
- **热重载**：开发时自动重载，实时查看效果
- **调试工具**：`console.log()` + `--log-console` 参数
- **FPS 监控**：QML 组件支持性能监控
